# Algorithmic Trading Platform  A personal project: fully automated multi-strategy options trading system for Indian equity derivatives.  ## What This Does - Connects to live market data via WebSocket (real-time OI, price, Greeks at sub-second latency) - Runs multiple automated trading strategies concurrently - Places and manages orders via broker REST APIs with smart order modification logic - Monitors positions with a live P&L dashboard served on AWS EC2 - Controlled remotely via Telegram Bot  ## Tech Stack - Language: Python (asyncio, threading, websockets, requests, pandas) - Cloud: AWS EC2 (24/7 production deployment) - Data: Angel One SmartAPI WebSocket for live market data - Execution: Dhan broker REST API for order management - Control: Telegram Bot API for remote commands and alerts ## Architecture Event-driven design: a single WebSocket connection streams live data and distributes to multiple strategy engines running in separate threads. Each strategy has an independent order manager with a background chase thread.  ## Instruments Traded Indian equity index options: NIFTY, SENSEX, BANKNIFTY, FINNIFTY (NSE/BSE FNO segment)  ## Note Strategy logic is proprietary and kept in a private repository.
